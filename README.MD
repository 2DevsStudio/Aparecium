## Description
API with spigot/paper support for creating GUIs, menus, etc. with ease.

## Functionalities
- Item system
  * Items with custom model data, nbt tags, enchants, lore, name, etc.
  * Items with custom textures (from url, file or resource)
  * ItemBuilder
  * ItemNBT
  * ItemNBTBuilder
  * ItemUtils
- Layout system
  *
- Layer system
- Command system
- Configuration system
- NBT system
- Sound system
- Chat system
- Event system
- Util system
- Reflection system
- Dependency system

## Supported Versions

1.12.2 <-> 1.19

Java 17+
